# EventHub High-Fidelity Dynamic Prototype

Це інтерактивний високодеталізований прототип для наступної лабораторної роботи на основі ЛР5 та ЛР6.

## Реалізовано

- головна сторінка EventHub;
- каталог подій;
- пошук і фільтри;
- сторінка події;
- додавання в обране;
- авторизація та форма реєстрації;
- стани помилки у формі;
- підтвердження участі;
- календар зареєстрованих подій;
- профіль з квитками;
- hover / active / error стани;
- повноцінна навігація між екранами.

## Кольорова гама

- `#0B1F3A` — основний темно-синій;
- `#2563EB` — пошук, фільтри, активна навігація;
- `#F97316` — CTA реєстрації;
- `#22C55E` — успіх, волонтерські/позитивні стани;
- `#F3F7FC`, `#FFFFFF`, `#D9E2EF` — фон, картки, межі.

## Запуск

Відкрити `index.html` у браузері або запустити локальний сервер у цій папці.
